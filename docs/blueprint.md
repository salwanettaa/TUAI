# **App Name**: TUAI

## Core Features:

- Crop Disease AI: Farmers can upload an image of a crop to receive AI-powered identification of diseases and grounded recommendations for treatment, stored in Firestore.
- Supply Chain Risk Advisor: An autonomous AI agent monitors global news, commodity prices, and supply events, then predicts supply shocks and recommends preventative actions for farmers.
- Intelligent Farmer Copilot: An interactive AI assistant (copilot) that provides instant answers to farmer queries, leverages weather data and crop records, and offers general advice or escalates complex plans to a more powerful AI.
- Local Supplier & Resource Finder: Users can search for nearby suppliers of agricultural inputs (e.g., fertilizer), and the AI will summarize the best options, locations, and directions.
- Farm Activity Log & Crop Records: Farmers can securely manage and view their farming activities, personal crop records, and operational data, with all results and user inputs stored in Firestore.
- Autonomous Alerts & Notifications: The AI-driven system proactively generates and pushes timely, region-specific alerts regarding potential risks, weather changes, and market fluctuations to farmers.
- Secure Phone-based Login: Provides farmers with a simple and secure authentication experience via their mobile phone numbers, including OTP verification.

## Style Guidelines:

- The chosen light color scheme conveys clarity and modernity, appropriate for an AI-powered agricultural tool. The primary color is a sophisticated deep blue (#1A6E98), symbolizing innovation, trustworthiness, and advanced technology. The background color is an extremely light, clean blue-grey (#F0F4F6), promoting an open and intuitive user experience. The accent color is a vibrant cyan (#59D2C6), offering a refreshing contrast for calls to action and highlighting key information while maintaining harmony with the primary hue.
- Headlines will use 'Space Grotesk', a sans-serif font, for a modern, tech-driven, and scientific feel, aligning with the AI focus of the application. Body text will utilize 'Inter', a grotesque-style sans-serif, ensuring optimal readability and a neutral, objective presentation of information. Code snippets or technical details will be rendered using 'Source Code Pro', a monospaced sans-serif, for clear differentiation.
- Use a set of modern, crisp, and functional outline icons. Icons should clearly represent agricultural concepts (e.g., crops, weather, irrigation), data analytics, and artificial intelligence, contributing to a clean and intuitive user interface.
- Adopt a clean, organized, and highly responsive layout, prioritizing the accessibility and clarity of information for mobile-first farmers. Utilize card-based designs for modular content presentation, ensuring an intuitive flow of critical insights and operational advice.
- Incorporate subtle and purposeful animations that effectively guide user attention and provide feedback, such as smooth transitions for loading AI analysis results or visual cues for alerts, without introducing distractions.
